# 🧠 Pneumonia Detection using Deep Learning

## 📌 Overview
This project implements a **Convolutional Neural Network (CNN)** to detect pneumonia from chest X-ray images. The model is trained to classify images into two categories: **NORMAL** and **PNEUMONIA**, demonstrating the application of Artificial Intelligence in healthcare diagnostics.

---

## 🚀 Features
- Deep learning-based image classification using CNN
- Medical image preprocessing and normalization
- Model evaluation using accuracy metrics
- Visualization of training performance

---

## 🛠 Tech Stack
- **Programming Language:** Python  
- **Libraries & Frameworks:** TensorFlow, Keras, OpenCV  
- **Data Processing:** NumPy, Pandas  
- **Visualization:** Matplotlib  
- **Machine Learning Tools:** Scikit-learn  

---
